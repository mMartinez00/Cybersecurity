# Cybersecurity Awareness Website

This project is a multi-page educational website designed to raise awareness about common cybersecurity threats and promote best practices for staying safe online. It presents key concepts in a clear, approachable way for non-technical users while maintaining a modern, professional design.

The site is structured to guide users from understanding threats, to learning protection strategies, and finally to exploring trusted resources and tools they can use in their everyday digital lives.

---

## Purpose

The goal of this website is to:
- Educate users about common cyber threats
- Encourage proactive cybersecurity habits
- Provide access to trusted official resources and optional security tools
- Demonstrate clean, consistent front-end design and layout practices

This project was built with clarity, accessibility, and visual consistency in mind.

---

## Pages Overview

### Home
Introduces the purpose of the website and sets the tone for cybersecurity awareness.

### About
Explains why cybersecurity matters, how users are affected by digital threats, and the importance of staying informed.

### Threats
Highlights common cybersecurity threats such as phishing, malware, and identity theft. Each threat is presented in an easy-to-understand format with clear descriptions and warning signs.

### Protection
Focuses on practical steps users can take to protect themselves online, including strong passwords, multi-factor authentication, safe downloads, backups, and user awareness.

### Contact
Provides contact information, a message form, official cybersecurity resources, and a curated list of optional personal security tools users can explore.

---

## Design Approach

- Consistent layout and spacing across all pages
- Reusable “glass card” UI components
- Dark, modern color palette with accessible contrast
- Responsive design for desktop and mobile screens
- Simple iconography and clear visual hierarchy

The website emphasizes readability and structure rather than technical complexity.

---

## Resources Included

The Contact page links to trusted official resources such as:
- FBI Internet Crime Complaint Center (IC3)
- Cybersecurity & Infrastructure Security Agency (CISA)
- Federal Trade Commission (FTC)

It also includes optional third-party tools for:
- Password management
- VPN and encrypted browsing
- Identity protection
- Secure cloud storage

These links are provided for educational purposes and do not represent endorsements.
